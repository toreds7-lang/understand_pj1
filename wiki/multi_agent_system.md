# Multi-Agent System
_Type: architecture | Papers: agent_survey_

## Summary
A Multi-Agent System (MAS) is a computational approach that involves multiple interacting agents operating within a shared environment. Each agent in a MAS can independently perceive its environment, make decisions, and execute actions to achieve individual or collective goals. Multi-Agent Systems are significant for solving complex problems that are difficult to address by a single agent due to their parallel, distributed nature and ability to model complex interactions.
