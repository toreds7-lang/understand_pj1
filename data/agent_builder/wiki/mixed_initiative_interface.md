# Mixed-Initiative Interface
_Type: concept | Papers: agent_builder_

## Summary
A Mixed-Initiative Interface is an interactive system where both human users and automated agents take turns to contribute to the progression of tasks. These interfaces allow for a dynamic exchange of control, enabling the system to propose actions or information and the user to accept, modify, or reject them. This concept is significant in designing systems that leverage both human intuition and computational power for robust decision-making and task execution.
