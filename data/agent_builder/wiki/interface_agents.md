# Interface Agents
_Type: concept | Papers: agent_builder_

## Summary
Interface agents are software agents that interact with users through various interfaces, typically designed to facilitate user-agency interactions by employing AI capabilities. These agents are significant as they provide a mechanism for users to interact with complex systems more naturally and efficiently, often enhancing user experience and system accessibility.
