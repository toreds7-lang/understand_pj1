# Vision Transformer
_Type: architecture | Papers: thinking_with_visual_primitives_

## Summary
The Vision Transformer (ViT) is an architecture that adapts the transformative power of transformers, originally designed for sequential data like text, to image processing tasks. It divides images into patches, treating these as sequences akin to words, to capitalize on transformers' strengths in sequence transformation. This has resulted in state-of-the-art performance in image classification and has opened new avenues for integrating transformer models in visual tasks.
