# Cold-Start Data
_Type: dataset | Papers: thinking_with_visual_primitives_

## Summary
Cold-Start Data refers to the initial set of data used by a machine learning model when no prior data is available. This is significant in scenarios where new systems—with no pre-existing data—are being developed or where they must operate effectively from the outset. The concept is critical for ensuring that models can begin learning effectively even in the absence of a large, labeled dataset.
