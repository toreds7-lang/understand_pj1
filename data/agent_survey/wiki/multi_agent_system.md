# Multi-Agent System
_Type: concept | Papers: agent_survey_

## Summary
A Multi-Agent System (MAS) is a framework where multiple autonomous agents interact within a shared environment to achieve individual or collective goals. These systems are significant in areas such as robotics, distributed computing, and artificial intelligence, where cooperative or competitive behaviors are modeled to solve complex problems. The collaborative model enhances flexibility, scalability, and reliability in handling tasks that a single agent might find challenging or impossible.
